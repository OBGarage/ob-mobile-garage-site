# OB Mobile Garage Website

## Run locally
1. Install Node.js
2. Open this folder in a terminal
3. Run:
   npm install
   npm run dev

## Deploy on Vercel
1. Go to Vercel
2. Create a new project
3. Upload this folder or import it from GitHub
4. Vercel should detect Next.js automatically
5. Deploy
